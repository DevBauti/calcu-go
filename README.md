## Calculadora simple go

### Calculadora simple de las 4 operaciones basicas.

-suma = add
-resta = subs
-multiplicacion = mult
-division = div

## Como ejercutar

### En consola:

    - > calcu-go.exe -add //suma
    - > calcu-go.exe -subs //resta
    - > calcu-go.exe -mult //multiplicacion
    - > calcu-go.exe -div //division
